IECONOMY

Simple Economy plugin for multiple server without MySQL, etc.

What is IEconomy?

IEconomy is a simple economy plugin with the support for multiple servers without the use of Databases and Vault. IEconomy works without Vault.

FEATURES

  • No MySQL needed -> no support
  • No custom messages
  • Vault support

How To Use

Installation

Open the zip file and move the IEconomy-Spigot.jar into the spigot plugins folder and the IEconomy-Sockets.jar into the sockets plugins folder and you are good to go.

Required Plugins

Requires Sockets and ILibrary with Server and Client enabled or else it wont work!
